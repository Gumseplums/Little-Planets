# Little Planets
Little Planets desklet by Gumseplums

## Features
#### Planet Type
Choose which type of cosmic body you want, aka the outline!
    - Planet
    - Planet with ring
    - Planet with two rings
    - Moon
    - Sun

#### Planet Colour
Of course you get to choose the colour. Gotta make sure it matches the setup!
    - Red
    - Orange
    - Yellow
    - Green
    - Blue
    - Purple
    - Pink
    - Brown
    - White
    - Gray
    - Black

#### Planet Texture
What kind of texture or look do you want the planet to have?
    - Plain
    - Shaded
    - Pixel (Big/Small)
    - Marble
    - Watercolour
    - Crayon
   

#### Special Interaction
What kind of special interaction does it have? Well you get to choose if these animations start and stop when Clicked. Start with click and stop after a set amount of time. Or are constantly happening.
    - Spin Clockwise
    - Spin Anti-Clockwise
    - Pulse
    - Bounce
    - Wiggle
    - Disappear
    - Reappear
    - Float
    - Change Size
    - None
